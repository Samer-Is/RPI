2. INTELLIGENT CATEGORY EXTRACTION
   Multi-layered approach to categorize competitor vehicles:
   
   A. Structured Extraction:
      - Attempts to extract vehicle cards with categories
      - Looks for keywords near prices (economy, luxury, suv, etc.)
      - Extracts vehicle brand/model names (Toyota, BMW, etc.)
   
   B. Price-Based Inference:
      When explicit category unavailable, uses price ranges:
      - 80-120 SAR → Economy
      - 120-160 SAR → Compact
      - 160-200 SAR → Standard
      - 200-250 SAR → SUV Compact
      - 250-300 SAR → SUV Standard
      - 300-350 SAR → SUV Large
      - 350-450 SAR → Luxury Sedan
      - 450+ SAR → Luxury SUV
   
   C. Fuzzy Keyword Matching:
      - Matches scraped text to Renty categories
      - Handles variations (e.g., "compact suv" matches "SUV Compact")
      - Cross-matches SUV and Luxury variants

3. CONFIDENCE SCORING SYSTEM
   Each competitor price gets a confidence score:
   
   - 95%: Exact category match from structured data
   - 85%: Keyword match with high relevance
   - 70%: Price-based inference with good alignment
   - 50%: Price-based inference with category mismatch
   
   Results sorted by:
   1. Confidence score (highest first)
   2. Price relevance to target category

4. ENHANCED DATA OUTPUT
   Each result now includes:
   - Competitor_Name: Source (Theeb, Hertz, Budget)
   - Competitor_Price: Price in SAR
   - Competitor_Category: Mapped category
   - Vehicle: Brand/model if extracted
   - Confidence: Match quality (50-95%)
   - Location: City where price is from
   - Date: Scrape timestamp
   - Source: Direct website source

HOW IT WORKS IN THE DASHBOARD:
-----------------------------

1. User selects branch (e.g., "King Khalid Airport - Riyadh")
2. User selects category (e.g., "Economy")
3. Scraper:
   a. Maps branch to location ("Riyadh Airport RUH")
   b. Scrapes 3 competitor sites (Theeb, Hertz, Budget)
   c. Extracts prices with categories
   d. Matches to target category
   e. Ranks by confidence
   f. Returns top 5 results
4. Dashboard displays:
   - Competitor comparison table
   - Average competitor price
   - Price chart with Renty vs competitors
   - Location and confidence indicators
